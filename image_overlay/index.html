Setup & Installation Instructions

1. Standard Install

  • Open the downloaded ImageOverlay.dmg file.
  • Drag the Image Overlay.app icon into your Applications shortcut folder.
  • Launch the app from Applications or Spotlight.

⚠️ "App is damaged" Warning

As this is a self-compiled utility without Apple Developer signature, macOS Gatekeeper may restrict it on first launch.

Quick Fix via Terminal (Recommended): xattr -cr /Applications/Image\ Overlay.app Alternative Fix:

Control-click (or right-click) the application, select Open from the context menu, then confirm.
